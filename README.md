# Attrify

A powerful attribute editor for Haiku operating system.

## Features
- Edit file attributes for any MIME type
- Support for all standard attribute types
- Create new files with predefined attributes
- Multi-window support
- Drag & drop support
- Internationalization

## Building
```bash
make
```

## Usage
Launch Attrify and either:
- Create a new Person file (File > New)
- Open an existing file (File > Open)
- Drag & drop a file onto the window
